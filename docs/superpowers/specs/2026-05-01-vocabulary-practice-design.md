# 小学生单词练习产品设计文档

## 概述

基于小学1-6年级1500个核心英语单词（按35+主题分类，含音标和emoji），构建一个Web端单词练习应用。目标是通过多种练习模式和科学的复习策略，帮助小学生强化单词记忆。

## 目标用户

- 小学1-6年级学生（独立使用）
- 家长（查看进度、管理设置）

## 技术栈

- 框架：React + TypeScript + Vite
- 样式：Tailwind CSS
- 数据存储：IndexedDB（通过 Dexie.js）+ localStorage
- 发音：Web Speech API（SpeechSynthesis）
- 部署：静态托管

## 页面结构

- 首页（角色选择）— 孩子入口 / 家长入口
- 孩子端：主题列表 → 学习模式（新词学习）→ 练习模式（选择题/拼写）→ 练习结果
- 家长端：密码验证 → 学习进度总览 → 主题详情/错词列表 → 设置

## 新词学习模式

孩子选择一个主题后，先进入学习模式浏览新词（该主题中等级为0、尚未练习过的词）：

- 每次展示一个词：英文 + 音标 + 中文 + emoji
- 自动播放发音，孩子可点击重复听
- 左右滑动或点击箭头切换上一个/下一个词
- 每次学习展示最多10个新词
- 浏览完毕后，进入练习模式对刚学的词进行测试

学习模式是轻量的"认识"环节，不做任何测试，目的是让孩子先对新词有初步印象。

## 练习模式

### 三种模式及切换逻辑

每个单词有掌握等级（0-5）：

- 等级0-1：看中文+emoji，从4个英文选项中选（认知阶段）
- 等级2-3：看英文单词，从4个中文选项中选（理解阶段）
- 等级4-5：看中文+emoji，手动输入英文拼写（产出阶段）

答对升一级，答错降一级（最低0）。等级5且连续3次答对标记为"已掌握"。

### 出题策略

每次练习出15题，按优先级混合：

1. 遗忘曲线到期词（最高优先，约40%）
2. 错题本高频错词（错2次以上且未掌握，约30%）
3. 当前主题新词（约30%）

某类词不够时，其他类自动补位。

### 选择题干扰项生成

- 优先从同主题词中选干扰项
- 避免选已掌握的词作为干扰项
- 拼写模式大小写不敏感，忽略首尾空格

### 即时反馈（克制风格）

- 答对：选项短暂变绿（0.5秒），轻微音效，自动进入下一题
- 答错：选项变红，高亮正确答案为绿色，停留1.5秒，然后进入下一题
- 拼写模式答错：显示正确拼写，错误字母标红对比

### 练习结束页

- 显示：答对X/15，用时X分钟
- 列出本次错词，每个可点击听发音
- "重练错词"按钮 + "返回"按钮
- 不做分数评级、星星等评价性展示

## 艾宾浩斯遗忘曲线

单词答对后，根据掌握等级安排下次复习时间：

| 等级 | 复习间隔 |
|------|----------|
| 0    | 立即（学习后直接进入练习） |
| 1    | 1天      |
| 2    | 2天      |
| 3    | 4天      |
| 4    | 7天      |
| 5    | 15天     |

- 答错时等级降一级，复习时间重置为对应等级间隔
- 已掌握的词（等级5且连续答对3次）进入30天长周期复习

## 错题本机制

- 答错自动记录，记录单词、错误时间、错误次数、错误类型
- 错题本中的词练习时优先出现，连续答对2次移出错题本
- 家长端可查看错题本列表

## 每日复习提醒

孩子进入首页时显示"今日待复习X个词"。有到期复习词时优先引导复习，但不强制，孩子也可选择学新主题。

## 家长端

### 进入方式

首页点击"家长入口"，输入4位数字密码。首次使用时设置密码，存在localStorage。

### 进度总览

- 已学单词数 / 1500
- 已掌握单词数
- 今日已练习词数 / 每日目标
- 总正确率
- 最近7天每日练习词数柱状图

### 主题详情

- 每个主题：已学/总数、掌握率
- 点进去看每个词的掌握等级
- 错词列表：按错误次数排序

### 家长可控设置

- 每日目标词数：默认15，范围10-30
- 主题解锁顺序：默认按顺序依次解锁（前一个主题80%的词达到等级1以上即解锁下一个），可手动解锁任意主题
- 难度调整：选项数量（3或4个）、拼写模式是否给首字母提示
- 重置功能：重置某主题学习进度（需二次确认）

### 不做的事

- 不做排行榜、社交对比
- 不做推送通知
- 不做复杂数据导出

## 数据模型

### 单词库（静态JSON）

```json
{
  "id": "fruit_apple",
  "english": "apple",
  "phonetic": "/ˈæpl/",
  "chinese": "苹果",
  "emoji": "🍎",
  "category": "水果篇",
  "categoryId": 7
}
```

从文章提取全部1500词，按35+主题分类，categoryId按文章顺序编号。

### 学习记录（IndexedDB）

```json
{
  "wordId": "fruit_apple",
  "level": 3,
  "consecutiveCorrect": 2,
  "nextReviewDate": "2026-05-08",
  "totalAttempts": 8,
  "totalCorrect": 6,
  "mastered": false,
  "lastPracticeDate": "2026-05-01"
}
```

### 错题记录（IndexedDB）

```json
{
  "wordId": "fruit_apple",
  "errorCount": 3,
  "consecutiveCorrect": 0,
  "lastErrorDate": "2026-05-01",
  "errors": [
    { "date": "2026-05-01", "mode": "spell", "userAnswer": "aple" },
    { "date": "2026-04-30", "mode": "choiceCnToEn", "userAnswer": "banana" }
  ]
}
```

### 用户设置（localStorage）

```json
{
  "parentPassword": "1234",
  "dailyGoal": 15,
  "unlockedCategories": [1, 2, 3],
  "choiceCount": 4,
  "spellHint": false,
  "unlockThreshold": 0.8
}
```

## UI风格

- 简洁清爽，大字体，适合孩子阅读
- 主色调柔和（浅蓝/浅绿），不用刺眼颜色
- 移动端优先响应式，手机操作流畅
- 按钮大、间距大，方便小手点击
